## 이슈
- 질문은 5~15분 정도 토론할 수 있는 것으로 준비한다.

- 가급적 서술형으로 답변할 수 있는 주제를 선정한다.

- 이미 먼저 등록된 이슈들과 서로 겹치지 않는 문제를 선정하여 이슈로 등록한다.

- 토론 목적의 문제 이외에도 자유롭게 책의 주제와 관련된 문제들을 등록할 수 있다.

- 세부 형식은 이슈 템플릿의 내용을 따른다.

- 코드를 실습했을 경우 코드도 올려도 무방.
